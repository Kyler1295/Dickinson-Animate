Dickinson Animation Guide


Colors

There are a list of 6 colors. 3 primary, 3 secondary.


  • Primary:
  • Red: var(--main-primary-red);
  • Gold: var(--main-primary-gold);
  • Black: var(--main-primary-black);

  • Secondary:
  • Yellow: var(--main-secondary-yellow);
  • Green: var(--main-secondary-green);
  • Blue: var(--main-secondary-blue);

Hover Effects

Each Hover as a small, medium, and large size (sm, md, lg).


Scale

Small Scale: add class: animate-hover-scale-sm


Medium Scale: add class: animate-hover-scale-md


Large Scale: add class: animate-hover-scale-lg




Rotate

45 Degree Rotate: add class: animate-hover-rotate-45


90 Degree Rotate: add class: animate-hover-rotate-90


180 Degree Rotate: add class: animate-hover-rotate-180





Pulse

Small Pulse: add class: animate-hover-pulse-sm


Medium Pulse: add class: animate-hover-pulse-md


Large Pulse: add class: animate-hover-pulse-lg





Tilt

Small Tilt: add class: animate-hover-tilt-sm


Medium Tilt: add class: animate-hover-tilt-md


Large Tilt: add class: animate-hover-tilt-lg





Shake

Small Shake: add class: animate-hover-shake-sm


Medium Shake: add class: animate-hover-shake-md


Large Shake: add class: animate-hover-shake-lg


Animations

Animations are just a constant animation. They also have small medium and large sub classes.

Medium Pulse: add class: animate-anim-pulse-md



Large Pulse: add class: animate-anim-pulse-lg






Tilt

Medium Tilt: add class: animate-anim-tilt-md



Large Tilt: add class: animate-anim-tilt-lg






Shake

Medium Shake: add class: animate-anim-shake-md



Large Shake: add class: animate-anim-shake-lg






Hover Animations (Border)

These are hover animations, specifically borders.


Primary Colors:

Red Border: add class: animate-anihover-border-primary-red



Gold Border: add class: animate-anihover-border-primary-gold



Black Border: add class: animate-anihover-border-primary-black



Secondary Colors:

Yellow Border: add class: animate-anihover-border-secondary-yellow



Green Border: add class: animate-anihover-border-secondary-green



Blue Border: add class: animate-anihover-border-secondary-blue



Fade Ins

Fade ins will add an animation when the object scrolls into screen. There are 4 types.


  • Fade Left
  • Fade Right
  • Fade Top
  • Fade Right

To add them add the following class to each perspectively.

  • scroll-animate animate-fade-left
  • scroll-animate animate-fade-right
  • scroll-animate animate-fade-top
  • scroll-animate animate-fade-bottom
